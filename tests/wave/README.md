# 量化交易策略集合

本目录包含基于中证1000指数的量化交易策略。

## 文件说明

### 1. 波段策略
- **strategy_wave_zz1000.py** - 波段理论策略
  - 基于多技术指标组合（MA、MACD、RSI、布林带）
  - 适合捕捉中短期波段行情
  - 回测前50只股票示例

### 2. 强势回踩策略
- **strategy_trend_pullbackV1.py** - 原始版本
  - 强势股回踩MA20支撑策略
  - 包含ATR动态止损、移动止盈
  - T+1开盘成交，避免未来函数

- **strategy_trend_pullback_optimized.py** - 优化版本（推荐）
  - 增强版本，包含详细日志和统计
  - 实时打印交易信息
  - 保存详细交易记录和资金曲线
  - 增加夏普比率等指标

### 3. 分析工具
- **analyze_trades.py** - 交易明细分析工具
  - 读取回测生成的CSV文件
  - 生成9张可视化图表
  - 详细的统计分析报告

### 4. 数据管理
- **data_manager.py** - 数据管理工具
  - 查看数据文件列表
  - 清理数据
  - 查看单只股票数据
  - 检查数据完整性

## 使用方法

### 运行波段策略
```bash
cd tests/wave
python strategy_wave_zz1000.py
```

### 运行强势回踩策略（推荐优化版）
```bash
cd tests/wave
python strategy_trend_pullback_optimized.py
```

**注意**: 首次运行会从接口获取数据并保存到 `stock_data/` 目录（每只股票一个CSV文件），后续运行会直接读取本地文件，大大加快速度。

### 管理数据
```bash
# 查看数据列表、清理数据
python data_manager.py
```

### 分析交易明细
```bash
# 先运行策略生成数据
python strategy_trend_pullback_optimized.py

# 再运行分析工具
python analyze_trades.py
```

## 数据存储机制

策略支持本地数据存储功能，避免重复调用接口：

1. **首次运行**: 从接口获取数据，保存到 `stock_data/000001.csv` 等文件
2. **后续运行**: 直接读取本地CSV文件，秒级加载
3. **增量更新**: 只下载缺失的股票数据
4. **禁用本地数据**: 在策略初始化时设置 `use_local_data=False`

### 数据文件格式

每只股票一个CSV文件，包含以下字段：
- 基础行情：open, close, high, low, volume
- 技术指标：ma10, ma20, ma60, atr14
- 选股指标：gain_20d, vol_surge, near_ma20

### 何时需要更新数据？

- 需要获取最新行情
- 股票代码列表变化（如成分股调整）
- 数据文件损坏

使用 `data_manager.py` 工具可以方便地管理数据。

## 策略参数

### 强势回踩策略参数
- **初始资金**: 1,000,000元
- **最大持仓**: 5只股票
- **单笔风险**: 总资产的2%
- **单票上限**: 总资产的30%
- **手续费**: 双边0.13%

### 选股条件
1. 20日涨幅 > 15%（强势股）
2. 收盘价 > MA20 > MA60（多头排列）
3. 近3日成交量放大1.5倍
4. 最低价触及MA20附近（回踩支撑）
5. 指数MA60过滤（只在牛市操作）

### 止损止盈
- **ATR止损**: 买入价 - 2×ATR14
- **移动止盈**: 从最高点回撤10%
- **趋势止盈**: 跌破MA10

## 输出文件

运行策略后会生成以下文件：

### 数据文件
1. **trade_log_detail.csv** - 详细交易记录
   - 包含每笔交易的买卖价格、持仓天数、收益率等

2. **equity_curve.csv** - 资金曲线数据
   - 包含每日总资产、现金、市值、持仓数量

3. **trade_analysis.png** - 可视化分析图表（运行analyze_trades.py后生成）
   - 资金曲线、回撤曲线、收益分布等9张图表

### 数据文件
- **stock_data/000001.csv** - 股票数据（每只股票一个CSV文件）
  - 自动生成，避免重复调用接口
  - 可直接用Excel/Pandas查看
  - 可使用 data_manager.py 管理

## 策略特点

### 优点
1. ✅ 避免未来函数（T日选股，T+1开盘成交）
2. ✅ 完善的风险控制（ATR动态止损+仓位管理）
3. ✅ 市场环境过滤（指数MA60过滤）
4. ✅ 科学的资金管理（Kelly公式变体）
5. ✅ 详细的交易记录和统计分析

### 注意事项
1. ⚠️ 数据源依赖百度接口，可能不稳定
2. ⚠️ 实际交易需考虑滑点
3. ⚠️ 需处理停牌情况
4. ⚠️ 手续费可能需要调整（实际约0.15-0.2%）
