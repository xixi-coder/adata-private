# 价值精选交易策略 V2 代码分析与说明

本文档基于 `value_strategy_v2.py` 当前实现整理，描述代码的实际行为与执行流程。

## 1. 策略目标与框架
- 目标：在价值因子和财务质量约束下，结合趋势与市场状态过滤，降低无效交易与亏损交易占比。
- 交易框架：
1. 每日检查持仓卖出条件（止盈/保本/趋势/估值）。
2. 按 `rebalance_period` 在调仓日筛选候选并买入。
3. 记录账户权益曲线并与沪深300基准对比。

## 2. 数据与缓存机制

### 2.1 数据来源
- 行情：`StockMarketBaiDu.get_market`。
- 财务：`Core.get_core_index`。
- 基准：`adata.stock.market.get_market_index(index_code='000300')`。

### 2.2 本地缓存
- `data/cache/kline/<code>.csv`：个股日线。
- `data/cache/finance/<code>.csv`：个股财务指标。
- `data/cache/benchmark_000300.csv`：沪深300。
- `data/cache/full_data_v2_5year.pkl`：整包缓存。

### 2.3 更新与兼容
- 行情缓存按最后交易日增量更新。
- 财务缓存 7 天内优先复用。
- 整包缓存命中时直接加载。
- 对旧缓存自动补齐技术列：`ma200`、`ma60`、`ma200_slope20`、`ret20`。
- 股票池过滤 `2/8/9` 开头代码。

## 3. 选股与交易规则

### 3.1 市场状态过滤（新）
- 当 `regime_filter=True` 时，仅在沪深300 `close > MA200` 时允许新增仓位。
- 大盘弱势期不新开仓，持仓仍按卖出规则管理。

### 3.2 买入条件（调仓日）
候选股需同时满足：
- 趋势约束：`price > ma60 > ma200`。
- 长期趋势向上：`ma200_slope20 > 0`。
- 中期动量：`ret20 > ret20_threshold`（默认 3%）。
- 价值与质量约束：
  - `EPS > 0`、`BPS > 0`
  - `PE < 25`
  - `PB < 3`
  - `ROE > 8`
  - 扣非净利润同比 > 0
  - 每股经营现金流 > 0
  - 资产负债率 < 70

### 3.3 候选评分
- 评分：`score = (500 / PE) + ROE + ret20 * 100`。
- 按分数降序补仓，直到 `max_positions`。

### 3.4 卖出规则（每日）
按优先级触发：
1. 固定止盈：`当前涨幅 >= take_profit_pct`（默认 12%）。
2. 保本止损：
   - 若历史峰值达到 `buy_price * (1 + breakeven_arm_pct)`（默认 6%），
   - 且当前价回落到 `buy_price * (1 + breakeven_buffer)`（默认 0.2%）附近，卖出。
3. 趋势卖出（确认版）：
   - 至少持有 `min_hold_days`（默认 15 天），且
   - 连续两日收盘价都在 MA200 下方。
4. 估值卖出：`PE > 60`。

### 3.5 交易执行细节
- 买入：剩余现金按剩余槽位等权分配。
- 股数：按 100 股整数手向下取整。
- 卖出：计入单边费率 `0.13%`。
- 冷却期（新）：卖出后同一标的在 `cooldown_days` 内不再买回（默认 20 天）。

## 4. 参数寻优（新）

### 4.1 寻优目标
- 主目标：最小化 `loss_ratio`（亏损交易占比）。
- 次目标：在亏损占比相近时最大化 `total_return`。
- 样本约束：优先选择 `closed_trades >= 60` 的参数组。

### 4.2 网格参数
- `rebalance_period`: `[10, 20]`
- `min_hold_days`: `[10, 15]`
- `cooldown_days`: `[10, 20]`
- `ret20_threshold`: `[0.02, 0.03]`
- `take_profit_pct`: `[0.10, 0.12]`
- `breakeven_arm_pct`: `[0.06]`

### 4.3 输出
- 参数搜索明细输出：`v2_param_optimization.csv`。

## 5. 回测输出与指标口径

### 5.1 输出文件
- `value_v2_trade_log.csv`：交易明细（含卖出原因、收益率等）。
- `strategy_metrics.csv`：核心绩效指标。
- `v2_strategy_report.png`：图形报告。
- `v2_param_optimization.csv`：参数寻优结果（新增）。

### 5.2 指标口径
- 年化收益率：按自然日年化。
- 最大回撤：基于 `total` 权益曲线。
- 胜率：仅统计已平仓交易。
- 夏普比率：`(日收益均值*252 - 0.02) / (日波动*sqrt(252))`。

## 6. 默认执行流程（主程序）
1. 计算近 5 年区间。
2. 预加载股票与财务缓存。
3. 运行参数寻优（以亏损占比最小为主）。
4. 用最优参数重跑回测。
5. 输出交易日志、指标、图表与寻优文件。

## 7. 注意事项
- 若使用 `local_only=True`，结果依赖本地缓存完整性与时效性。
- 买入端未显式加入滑点和买入手续费，实盘需补充。
- 代码中存在裸 `except`，可能隐藏数据异常原因，建议后续补日志。
