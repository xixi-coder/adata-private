# 趋势交易策略

`TrendTradingStrategy` 是供每日选股任务使用的日线趋势扫描器，复用统一的 A 股票质过滤，并输出两类候选：

- 趋势突破：均线多头排列、MA60 向上、放量突破前 20 日高点，同时限制单日涨幅和距 MA20 的乖离。
- 趋势回踩：上升趋势未破坏，盘中温和回踩 MA20 后收回，排除放量下跌。

候选包含趋势评分、风险等级、观察价和失效价。策略入口为 `jobs/trend/run_daily.py`，默认随统一调度的 `eod` profile 执行。

目录中的 `trend_strategy_code.py` 是原有的周趋势 + 基本面增强回测，继续保留供历史研究使用。
