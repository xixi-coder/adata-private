# 股票分析 Skill

对股票进行系统化、多维度的分析，并输出结构清晰、有依据、有风险提示的结论。

> 适用范围：单只或多只股票的**个股深度分析**。若用户提供的是**整个持仓组合 / 持仓截图 / 账户明细**，需要做组合级的仓位结构、集中度、赛道敞口与盈亏归因分析，请改用 `portfolio-analysis` skill。

## 核心原则

1. **数据驱动**：所有结论必须有数据或事实支撑，不臆测、不编造数字。拿不到的数据要明确说明「数据缺失」，绝不虚构。
2. **多维印证**：单一指标不构成结论，需要多个维度相互印证后再下判断。
3. **风险优先**：先讲清楚风险，再谈机会。任何分析都必须包含风险提示。
4. **不做投资建议**：输出的是「分析」而非「买卖指令」。用「关注/警惕/观察」等中性表述，避免「必涨/建议买入」等承诺性语言，并在结尾附免责声明。

## 分析流程

按以下步骤推进，缺失的数据维度要标注清楚，不要跳过风险环节。

### 1. 明确目标
- 确认股票代码 / 名称、市场（A股/港股/美股）。
- 确认分析目的：长期价值投资、中短期交易、事件驱动、还是横向对比。
- 确认时间视角：短线（数日~数周）、中线（数月）、长线（数年）。

### 2. 数据采集
优先使用本仓库的 `adata` 库获取 A 股数据（见下方「数据获取参考」）。若数据不可得，说明来源缺失，不要编造。

需要采集的维度：
- **基础信息**：所属行业、总股本/流通股本、市值。
- **行情数据**：近期 K 线（开高低收、成交量）、涨跌幅、区间振幅。
- **财务数据**：营收、净利润、毛利率/净利率、ROE、资产负债率、经营性现金流、同比/环比增速。
- **资金面**：主力资金流入流出、北向资金、成交额变化。
- **估值**：PE、PB、PS，与行业均值及自身历史分位对比。

### 3. 多维度分析

**基本面**
- 盈利能力：营收/净利润趋势、毛利率与净利率、ROE 水平及变化。
- 成长性：近几期同比、环比增速是否稳定、是否加速或放缓。
- 财务健康：资产负债率、流动/速动比率、现金流是否与利润匹配（警惕增收不增利、利润远高于现金流）。
- 行业地位：所处行业景气度、公司在行业中的相对位置。

**技术面**
- 趋势：均线排列（多头/空头/纠缠）、价格所处位置。
- 量价关系：放量上涨/缩量下跌等配合关系，是否量价背离。
- 关键位：支撑位、压力位、近期高低点。
- 动能：涨跌节奏、是否超买超卖。

**资金面**
- 主力资金近 N 日净流入/流出趋势。
- 成交额是否放大、换手率是否异常。
- （A股）北向资金、龙虎榜等异动。

**估值面**
- 当前 PE/PB 处于自身历史的什么分位。
- 与同行业可比公司横向对比，判断高估/低估/合理。

**消息面（如适用）**
- 近期公告、业绩预告、行业政策、突发事件的影响。

### 4. 综合结论
- 给出多空因素清单（利好 / 利空分开列）。
- 给出综合判断：强/中/弱，及主要理由。
- 明确适用的时间视角和前提条件。

### 5. 风险提示（必须）
- 列出该股票当前主要的下行风险。
- 说明分析所依赖数据的时效性与局限性。
- 附免责声明：本分析仅供参考，不构成投资建议，市场有风险，决策需谨慎。

## 输出格式

用中文输出，结构建议如下：

```
# 股票分析：<名称>（<代码>）

## 一、概览
- 市场 / 行业 / 市值 / 分析视角

## 二、基本面
（盈利、成长、财务健康、行业地位）

## 三、技术面
（趋势、量价、关键位）

## 四、资金面
（主力资金、成交、异动）

## 五、估值
（PE/PB 分位、同业对比）

## 六、综合结论
利好因素：
- ...
利空因素：
- ...
综合判断：<强/中/弱> + 理由 + 适用视角

## 七、风险提示
- ...
> 免责声明：本分析仅供参考，不构成投资建议。市场有风险，决策需谨慎。
```

多股对比时，用表格横向罗列关键指标（估值、成长、ROE、资金流等），再给出排序和理由。

## 数据获取参考（adata 库）

本仓库自带 `adata`，可用于获取 A 股数据。常用接口：

```python
import adata

# 基础信息
adata.stock.info.all_code()                          # 全部股票代码表
adata.stock.info.get_stock_shares(stock_code='000001')  # 股本信息
adata.stock.info.get_industry_sw(stock_code='000001')   # 申万行业

# 行情（日K等）
adata.stock.market.get_market(stock_code='000001', k_type=1, start_date='2024-01-01')

# 财务核心指标（营收、利润、ROE、负债率、现金流等）
adata.stock.finance.get_core_index(stock_code='000001')

# 资金流向
adata.stock.market.get_capital_flow(stock_code='000001')
```

> 具体接口参数以仓库实际实现为准。若某接口返回空或报错，说明数据源暂不可用，在结论中标注数据缺失，不要用假数据填充。

## 注意事项

- 涉及具体买卖点位时，用「关注区间 / 参考位」而非「买入价」。
- 不承诺收益、不预测确定性涨跌。
- 数据有明确日期时，标注数据截止日期，避免用过期数据下结论。
