# 趋势交易选股

盘后从共享 A 股日线缓存的全部普通 A 股中扫描两类候选，不再限制前 2500 只；ST、退市风险股票仍会被过滤。

- 趋势突破：`close > MA20 > MA60 > MA120`，60 日线向上，放量突破前 20 日高点。
- 趋势回踩：多头排列保持，盘中回踩 MA20 后收回，且没有出现放量下跌。

策略先复用统一票质门槛，过滤 ST/退市风险、上市不足、流动性不足和近期结构破坏的股票；候选再按趋势强度、量价确认和风险水平排序。

运行：

```bash
python jobs/trend/run_daily.py
```

可通过 `TRADE_DATE=YYYY-MM-DD` 指定信号日期。结果写入 `jobs/trend/outputs/`。

默认不限制股票池数量。如需调试小样本，可设置 `TREND_UNIVERSE_SIZE=100`；设置为 `all`、`none` 或 `0` 仍表示全部股票。
